# Frontend Design

## Principles

- Pick a clear aesthetic direction (minimal, brutalist, editorial, etc.)
- Typography and color with intent — avoid default purple gradients
- Responsive and accessible by default
- Match existing project design tokens when present

## Process

1. Read existing components and styles
2. Define layout, type scale, color, motion
3. Implement with semantic HTML and focus states
4. Verify in browser when possible

Load `design-taste-frontend` or `anti-ai-slop` skills for visual quality checks on substantial UI work.
