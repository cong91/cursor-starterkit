# Research

Research a topic with authoritative sources first.

## Source hierarchy

1. Official docs, type definitions, source code
2. Official blogs, changelogs, specs
3. Browser compat, release notes
4. Stack Overflow, blogs — verify before citing

## Process

1. State what you need to verify
2. Search/read Tier 1-2 sources
3. Cross-check conflicting claims
4. Summarize with links; label inferences as inferences

For real products (SDKs, devices, APIs), web search current state — do not guess versions or launch dates.
