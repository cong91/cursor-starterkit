# Resume

Resume work on a project after a gap.

## Steps

1. Read `.cursor/memory/project/state.md` for current focus + next steps
2. Run `/memory-search --recent` to recall recent observations
3. `br list` if beads is active — see reserved/open tasks
4. Summarize: "Last we were doing X, next is Y"
5. Confirm with the user before starting

If state.md is empty/stale, suggest `/init` to re-detect.
