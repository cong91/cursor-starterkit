# Status

Report current project status concisely.

## Gather

- `.cursor/memory/project/state.md` — focus + blockers
- `git status` — uncommitted changes
- `br list` — open beads tasks (if available)
- `/memory-search --recent` — recent decisions

## Output

```
Focus: <one line>
Uncommitted: <N files>
Open tasks: <br count or none>
Recent decisions: <top 2>
Blockers: <none / list>
```
