# UI Review

Review UI changes for quality, accessibility, and "AI slop".

## Steps

1. Screenshot the changed UI (Playwright/chrome-devtools MCP, or ask user)
2. Load `anti-ai-slop` + `accessibility-audit` skills
3. Check: distinctive design, semantic HTML, contrast, focus states, responsive
4. Report findings grouped: 🔴 must fix / 🟡 suggestion / 🟢 nice to have
5. Cite `file:line` for each finding

Don't claim the UI is good without seeing it rendered.
