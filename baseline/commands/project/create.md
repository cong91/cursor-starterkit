# Create

Spec-first feature creation. Define what to build before implementing.

## Output

A short spec written to `.cursor/memory/project/state.md` (or a draft):

```markdown
## Feature: <name>
- Goal: <one sentence>
- Acceptance: <bullet criteria>
- Out of scope: <explicit>
- Files likely touched: <guesses>
```

## Then

Suggest `/plan` for implementation planning, then `/start` to begin.
