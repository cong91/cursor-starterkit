# Memory Templates

Project memory lives in `.cursor/memory/project/`. Templates for new sections are in `_templates/`.

## Files

| File | Purpose |
| ---- | ------- |
| `project.md` | Purpose, structure, overview |
| `tech-stack.md` | Stack, scripts, verification commands |
| `state.md` | Current focus, blockers |
| `roadmap.md` | Now / next / later |
| `user.md` | Preferences |
| `gotchas.md` | Footguns |

Initialized by `csk install`. Refined by `/init` in Cursor.
